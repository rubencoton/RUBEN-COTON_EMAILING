# RUBEN-COTON_EMAILING

Sistema de email marketing para RUBEN COTON.

## Objetivo

Gestionar campañas de emailing: plantillas HTML, listas de contactos, envíos y seguimiento.

## Estructura

```
RUBEN-COTON_EMAILING/
├── README.md
├── .claude/
│   └── CLAUDE.md           # Contexto para Clauder
├── templates/               # Plantillas HTML de correos
├── assets/
│   ├── css/                 # Estilos inline-ready para email
│   └── img/                 # Imágenes para campañas
├── campaigns/               # Correos finales listos para enviar
└── config/                  # Configuración del sistema
```

## Stack

- HTML + CSS inline (compatibilidad email)
- Ejecución local
